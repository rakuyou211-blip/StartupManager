# Changelog

## v1.5.0 (2026-07-12)

### 追加
- 一覧に実行ファイルのアイコンを表示
- exe/ショートカットを一覧へドラッグ&ドロップすると入力済みの追加ダイアログが開く
- 新規追加でスタートアップフォルダ(ショートカット)方式を選択可能に
- 実行中プロセスの検出(ツールチップ・詳細に表示)
- 右クリック「無効の項目をすべて選択」「リンク切れの項目をすべて選択」
- タスク項目のトリガー種別(ログオン時/システム起動時)を詳細に表示
- セルフテストを22項目に拡張: ストアアプリの状態切替往復(元の値へ復元)、実行中プロセス検出

## v1.4.0 (2026-07-12)

### 追加
- ストアアプリ(UWP)のスタートアップタスクに対応: 列挙と有効化/無効化(タスクマネージャーと同じ管理範囲に)
- 右クリック「この名前をWebで検索」
- ウィンドウサイズ・「システムのタスクも表示」を `settings.json` に記憶
- セルフテストを19項目に拡張: スタートアップフォルダ項目の往復、
  完全削除→バックアップ復元のエンドツーエンド往復、CSVエクスポート、UWP列挙

### 改良
- 一覧更新中はウェイトカーソルを表示
- 「定義元を開く」がストアアプリではWindows設定のスタートアップアプリ画面を開くように

## v1.3.0 (2026-07-12)

### 追加
- 新規追加ダイアログ: 実行ファイルを選んでレジストリRun(HKCU/HKLM)に起動項目を登録
- `-SelfTest` モード: ダミー項目で列挙→無効化→再有効化→後片付けを自動検証(13項目)
- 詳細表示に「無効化した日時」を追加(StartupApprovedのFILETIMEを解読)
- 行ツールチップ(フルコマンド/実行ファイル/リンク切れ警告)
- ステータスバーに選択件数を表示
- 非管理者時の「管理者として再起動」リンク
- 多重起動防止(Mutex)

### 改良
- ListViewのちらつき防止(ダブルバッファリング)
- Deleteキーが未選択時に警告を出さないように

## v1.2.0 (2026-07-12)

### 追加
- 発行元列(実行ファイルの会社名)+ 詳細にデジタル署名の有無とバージョン
- リンク切れ検出: 実行ファイルが存在しない項目を赤字表示・件数集計
- バックアップからの復元ダイアログ(.reg取り込み / ファイル書き戻し / タスクXML再登録)
- 右クリック「定義元を開く」(regeditジャンプ / タスクスケジューラ / フォルダ)
- キーボードショートカット(F5 / Ctrl+A / Delete / Enter / Esc)
- `-Backup` CLIスイッチ

### 改良・修正
- 列挙結果をキャッシュし、検索入力のたびにタスクスケジューラへ照会しないように(高速化)
- 件数表示をステータスバーに分離(ボタン列に隠れる問題の修正)
- ショートカット解決のCOMオブジェクトをループ外で1回だけ生成
- Meiryo UIフォント適用、ウィンドウ幅拡大

## v1.1.0 (2026-07-12)

### 追加
- 列見出しクリックでソート(昇順/降順)
- 右クリックメニュー(有効化/無効化/場所を開く/コマンドコピー/詳細/完全削除)
- ダブルクリックで詳細表示
- CSVエクスポート(GUIボタン + `-Export` CLI)
- タイトルバーにバージョン表示

### 改良・修正
- 検索を大文字小文字を区別しない部分一致に(記号を含む検索の誤動作を修正)
- 完全削除時のバックアップ先参照をスクリプトスコープに修正

## v1.0.0

- 初版: レジストリRun(HKCU/HKLM/32bit)・スタートアップフォルダ・ログオン/起動タスクの一覧、
  有効化/無効化(StartupApproved方式)、完全削除(自動バックアップつき)、`-List` モード
