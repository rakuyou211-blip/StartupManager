# 変更のきろく

落葉いが、StartupManager をどう直してきたかの、正直なメモです。新しいものが上にあります。
（日付が同じ日に固まっているのは、最初の数日で一気に形にしたためです。）

---

## v1.11.1 — 読みものと、配布の整理（2026-07-13）

- この「変更のきろく」を、読みやすく書き直しました（できることは v1.11.0 のままです）。
- ダウンロードのZIPから、使うのに要らないファイル（宣伝用のメモや、私の手元用スクリプト）を外して、すっきりさせました。中身は本体と説明書だけです。

## v1.11.0 — 見た目とデータを少し整える（2026-07-13）

- README に **画面のスクリーンショット** を載せました。表示はサンプルで、実在のソフトやあなたの情報は写っていません。
- 一覧を書き出す **CSV に「発行元」の列** を足しました。誰が作ったソフトかも、まとめて確認できます。

## v1.10.0 — こわくない削除のために（2026-07-13）

- **大事そうなものを消す前に、一声かける**ようにしました。セキュリティソフト、音・画面・入力のドライバー、クラウド同期っぽいもの…を「完全削除」しようとすると、「これは大事かもしれません」とやさしく止めます。断定はしません。迷ったら、まず「無効化（あとで戻せる）」を。
- 詳細に **ファイルの更新日時** を出すようにしました。「いつ入れたか覚えてないやつ」を見分ける手がかりになります。

## v1.9.0 — 掃除の的をしぼる（2026-07-13）

- **「サードパーティのみ」表示** を足しました。Windows標準やMicrosoftのものを隠して、「あとから自分で入れたもの」だけを一覧にできます。どれを掃除の対象にするか、迷いにくくなります。

## v1.8.0 — 復元まわりを、本気で直した（2026-07-12）

このバージョンは、大きな節目です。コードを何人もの目で読み直して（自動レビュー）、**バックアップと復元の危ない不具合をまとめて直しました。** 「消したのに戻せない」が、いちばん起きてほしくないことなので。

直した主なもの：

- 完全削除のたびに、ちゃんと新しいバックアップを取るように。（前は最初の1回だけで、2件目以降が戻せませんでした。）
- 削除したタスクを復元したとき、**元のフォルダの場所まで正しく戻す**ように。
- 管理者権限がないのに「削除しました」と表示していたのを、ちゃんと「できませんでした」と正直に伝えるように。
- 新規追加のダイアログで、選択が取り違えられて別の場所に登録される不具合を修正。
- （macOS）バックアップに失敗した項目は削除しない／名前に記号が入っても壊れない、など安全側に。
- 一覧に「起動場所」の列も追加しました。

## これまでの歩み（v1.0 〜 v1.7・2026-07-12）

最初の数日で、土台と日々の使い勝手を一気に入れた時期です。

- **v1.0** — はじまり。レジストリRun・スタートアップフォルダ・タスクの一覧、有効化/無効化、削除（バックアップつき）。
- **v1.1〜v1.3** — 並べ替え・右クリックメニュー・詳細表示・CSV出力・新規追加・自己テスト。使うたびの小さな不便を減らしていきました。
- **v1.4〜v1.5** — ストアアプリ対応、アイコン表示、ドラッグ＆ドロップでの追加、いま動いているかの表示。
- **v1.6** — **macOS版**（`StartupManager.command`）が仲間入り。完全オフラインで動くことを、はっきり書きました。
- **v1.7** — macOS版にも自己テストとCSVを入れて、Windowsと機能をそろえました。F1ヘルプや、配布用ZIPを作るスクリプトも。
