# StartupManager

PC起動時に自動起動するソフト/アプリを一覧表示し、**無効化・有効化・完全削除**できる Windows 用 GUI ツールです。
PowerShell スクリプト1本で動作し、インストール不要です。

## 対象となる起動項目

| 種類 | 場所 |
|---|---|
| レジストリ Run キー | HKCU / HKLM / HKLM(32bit, Wow6432Node) |
| スタートアップフォルダ | 現在のユーザー / 全ユーザー |
| タスクスケジューラ | ログオン時 / 起動時トリガーのタスク |

無効化は Windows のタスクマネージャーと同じ仕組み(`StartupApproved` レジストリ)で行うため、**いつでも元に戻せます**。

## 使い方

1. このリポジトリをダウンロード(Code → Download ZIP)して展開
2. `StartupManager.bat` をダブルクリック(UAC で管理者昇格します)
3. 一覧から項目を選んで「有効化 / 無効化 / 完全削除」

> 管理者権限なしでも起動できますが、HKLM(全ユーザー)側の項目は操作できません。

### コマンドラインモード

```powershell
# GUIなしで一覧をテキスト表示
powershell -NoProfile -ExecutionPolicy Bypass -File StartupManager.ps1 -List

# CSVに書き出し
powershell -NoProfile -ExecutionPolicy Bypass -File StartupManager.ps1 -Export items.csv

# バックアップだけ作成
powershell -NoProfile -ExecutionPolicy Bypass -File StartupManager.ps1 -Backup
```

## 機能

- 起動項目の一覧表示(状態 / 名前 / 種類 / **発行元** / コマンド)
- 有効化 / 無効化(タスクマネージャー互換の可逆操作)
- 完全削除(削除前に自動でバックアップを作成)
- **バックアップからの復元**(GUIから日時を選んでワンクリックで書き戻し)
- **リンク切れ検出**: 実行ファイルが存在しない項目を赤字で表示(アンインストール残骸の掃除に)
- 絞り込み検索(大文字小文字を区別しない部分一致。発行元も検索対象)
- 列見出しクリックでソート(再クリックで昇順/降順切替)
- 右クリックメニュー(有効化 / 無効化 / ファイルの場所を開く / **定義元を開く** / コマンドをコピー / 詳細 / 完全削除)
- ダブルクリックで詳細表示(実行ファイルのパス・発行元・バージョン・**デジタル署名の有無**など)
- CSV エクスポート(GUI / コマンドライン両対応)
- システムタスク(`\Microsoft\` 配下)の表示切替
- キーボードショートカット: `F5` 更新 / `Ctrl+A` 全選択 / `Delete` 無効化 / `Enter` 詳細 / `Esc` 検索クリア

## バックアップと復元

- 完全削除の前に、レジストリ(.reg)とスタートアップフォルダの内容が `Backups\日時\` に自動保存されます
- 「バックアップ作成」ボタンでいつでも手動バックアップできます
- 復元方法:
  - レジストリ: バックアップ内の `.reg` ファイルをダブルクリックして取り込み
  - スタートアップフォルダ: `removed_～` ファイルを元のフォルダに戻す
  - タスク: `task_～.xml` をタスクスケジューラの「タスクのインポート」で取り込み

## 動作環境

- Windows 10 / 11
- Windows PowerShell 5.1(標準搭載。追加インストール不要)

## 注意事項

- レジストリとタスクスケジューラを変更するツールです。削除前バックアップは自動で作成されますが、利用は自己責任でお願いします
- ウイルス対策ソフトによっては、レジストリを操作するスクリプトとして警告される場合があります
